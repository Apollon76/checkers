Checkers

	Created by Nikolay Permyakov.

To perform this program on your computer python3 must be installed. You can download it on http://python.org.

	Starting
On Linux:
Go to the root of project.
Write in terminal:
$ python3 Main.py
For running in console mode:
$ python3 Main.py -c

On windows:
Double click on Main.py.
For running in console mode:
python3 Main.py -c
